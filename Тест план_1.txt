Тест план VendorMachine
Инструкция – Проведение тестирования программы VendorMachine, предназначенной для эмуляции работы автомата по продаже напитков 
Инструменты для тестирования 
- YouTrack
-TestRail
Тестовое оборудование 
PC – ПроцессорIntel core i5 9400f, 2.90 GHz, Видеокарта GTX 1660 Ti Оперативная память 2x16 Gb DDR-4, Накопители 240 Gb SSB, 1Tb HDD, Операционная система OS Windows 10 Pro сборка 19042
Объем работы
Необходимо протестировать
- графический интерфейс 
- выполнение заявленных функций 

Что будет протестировано 
UI
- отображение всех необходимых элементов для выбора объема напитка
- корректность отображение цен и объема
- корректное отображение уникальной иконки для каждого товара 
- корректное отображение заказа
- проверка орфографии 
- корректное отображение данных в отчете оплаты
-корректное отображение данных на странице оплаты
- возможность ввода промокода 

Функциональное тестирование
-возможность заказа напитка 
-ввод денежных средств 
- удаление позиций в заказе
- проверка на ограничение заказа
- проверка расчета при активации промокоде 
- ограничение ввода денег при достаточном количестве для оплаты   

Что не будет протестировано
- кроссплатформенность приложения
- нагрузочное тестирование
