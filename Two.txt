Днс это система доменных имен. каждый сайт имеет свой IP адрес, чтобы попасть на нужный нам сайт мы должны знать IP адрес данного сайта. Для облегчения этого процесса была создана днс система. IP адрес получает свое уникальное имя (название сайта) 
HTTP/HTTPS это два протокола, с помощью которых передается информация в интернете. Передача происходит между клиентом и сервером. HTTP не имеет шифрование данных, используется там где безопасность не имеет значение. HTTPS  имеет шифрование данных используется для безопасности данных ( в онлайн магазинах, кошельки, гос сайти и другое)  
Версии
HTTP:
	HTTP 0.9 - Экспериментальная версия протокола обмена посредством протокола TCP, реализующего службу клиент-сервер
	HTTP 1.0 - Первая официальная версия протокола
	HTTP 1.1 - Протокол, позволяющий отправлять сразу несколько запросов в течение одного соединения по TCP.
	HTTP 2.0 - Бинарная версия протокола, поддерживающая расстановку приоритетов запросов и их мультиплексирование
